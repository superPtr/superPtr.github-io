Common Commands need to know for me in Linux (Keep Updating)

Based on my cloud server (ESC) using CentOS, learning Linux commands is necessary.

Therefore, I am sorting out the common commands that I used before or learned before in this post.

Basic Concept

[root@abc]# [usr01@abc]$

The concept of root is similar as Windows’ administrator, and usr01 is similar as general user with lower rights

“#” is the symbol for root user

“$” is the symbol for general user

“/“ means root directory, similar as Windows “This PC”
